Troubleshooting Scheduled Tasks

If a scheduled task didn't run:

  1. Check Beat Status - Verify Celery Beat scheduler is running
  2. Check Task Monitor - Look for task locks or errors
  3. Run Manually - Use "Run Now" button to execute immediately
  4. Check Logs - Review Celery worker logs for errors
  5. Verify Time - Ensure server time is correct (UTC)

Common Issues:

  • Celery Beat not running - Restart the beat scheduler
  • Worker offline - Check if Celery workers are active
  • Task lock stuck - Check Task Monitor for stale locks
  • API rate limits - DataGolf API may be rate-limited
